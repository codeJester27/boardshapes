# Boardmesh

To run a Go project:

- [Install Go](https://go.dev/dl/)

- Depending on your project, enter one of the following

  - `cd WebApp`
  - `cd DiscordBot`

- Set environment variables. Ask Jared.

- Enter in terminal:
  ```ps
  go get .
  go mod tidy
  go run .
  ```
